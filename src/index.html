Rock Paper Scissors game

Maintainable and extendable JS version of popular game

Features

Main goal of the app is extensibility and maintainability following a simple and re-usable architecture

  • unlimited number of players ( limited only by UI as of now )
  • eliminations mode ( check "Player vs 10 Computers" game )
  • component based architecture ( re-usable and standalone code )
  • easiness of adding new stages ( see core/game/game.js )
  • easiness of adding new gestures ( add in a single place: config/game-modes.js )
